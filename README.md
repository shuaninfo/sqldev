## 功能

- SQL 
    - DQL 
    - DML
    - 数据导出
    - 自动补全，智能提示 
    - 查询语句审计
    - 异步查询
    - 动态脱敏
- SQL审核
    - 流程化工单（待实现）
    - SQL语句检测与执行
    - SQL回滚（待实现）
    - 历史审核记录
- 推送
    - E-mail推送   
- 用户权限及管理
    - 多角色
    - 细粒度权限到字段级别
    - 查询、修改、导出可独立赋权
- 适配
    - 引擎：Oracle、Mysql、MongoDb、MSsql
    - 环境：自有机房、阿里云、aws、腾讯云     
- 集成
    - CAS登录 
    - LDAP登录 
    - OAUTH2登录
